== Description ==

Customizable chatbot for businesses powered by Allio Conversational AI engine and Dialogflow.
Enables businesses to create their own branded chatbot, and to expose services to customers.
Provides shortcode and widget are to display the chatbot.

== Installation ==

Contact sales@allio.io to obtain a license key and an engine token.
Install the plugin and activate it.
Go to Allio Settings menu, enter the license key, engine token and logo.
Save
Add the [allio_chatbot] shortcode inside the contents of a page
View your page and engage in conversation with your chatbot.
